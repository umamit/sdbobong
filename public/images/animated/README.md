# Folder Icon Animasi

Masukkan file icon animasi (format `.gif` atau `.webp` dari Flaticon) ke folder ini dengan nama-nama berikut:

### 1. Dapodik Bento Grid (Halaman Profil)
- `school.gif` (untuk Nama Resmi Sekolah 🏫)
- `key.gif` (untuk NPSN 🔑)
- `government.gif` (untuk Status Sekolah 🏛️)
- `calendar.gif` (untuk SK Pendirian 📅)
- `shield.gif` (untuk Akreditasi 🛡️)
- `book.gif` (untuk Kurikulum 📖)
- `pin.gif` (untuk Alamat 📍)
- `land.gif` (untuk Status Lahan 🌳)

### 2. Langkah PPDB (Halaman Utama)
- `info.gif` (untuk Langkah 1: Pelajari Info)
- `form.gif` (untuk Langkah 2: Isi Formulir)
- `download.gif` (untuk Langkah 3: Unduh Berkas)
- `chat.gif` (untuk Langkah 4: Hubungi Operator)

### 3. Karya & Prestasi (Halaman Kesiswaan & Profil)
- `trophy.gif` (untuk Juara/Medali/Prestasi 🏆)
- `art.gif` (untuk Seni/Karya Lukisan 🎨)
- `craft.gif` (untuk Prakarya/Bambu 🪵)
- `poetry.gif` (untuk Puisi/Tulisan 📝)
